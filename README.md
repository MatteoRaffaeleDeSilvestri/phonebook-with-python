# Software of a phonebook with Python

ContactEase Solutions aims to simplify the management of telephone contacts for its users by developing an intuitive and interactive software that optimizes the organization and access to personal information.

Users often find it difficult to manage and organize their telephone contacts efficiently. There are few simple and intuitive solutions that allow you to add, edit, delete, view and search contacts in one place, directly from the terminal.

ContactEase Solutions will provide an interactive console application which, thanks to the principles of object-oriented programming (OOP) in Python, will enable simple and structured contact management. Users can easily save and upload contacts in a file format (such as JSON), ensuring efficient and secure data management.

**Project requirements:**

1. **OOP in Python:** Implement OOP concepts for a solid structure and
scalable.
2. **Data Structure:** Create an efficient data structure to store contacts.
3. **User Interface:** Develop an interactive and easy-to-use command line interface.
4. **Functionality:**
    * **Adding a Contact:** Allow new contacts to be added.
    * **Contacts view:** Show all existing contacts.
    * **Editing a Contact:** Allow the details of existing contacts to be edited.
    * **Deleting a Contact:** Remove contacts from the address book.
    * **Contact search:** Search contacts by first or last name.
    * **Save and Upload:** Save contacts to a file and load them on startup.

**User Interface:** The interface will be command-line based, offering a main menu with clear options for various operations, thus ensuring a smooth and accessible user experience even for novice users.
